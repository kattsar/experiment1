Damian_et_al_2011.rmd : exercise to understand how to analyse data similar to the ones I would get from REPSWITCH1 (based on the paper by Damian et al., 2011).

Keyboard_covert_files.rmd : code to create the text files needed torun the code on the comparison of the two keyboards

PNG files : result figures from the two pilots 

Pilot1_Analysis.rmd & Pilot2_Analysis.rmd : code for analysing the data from pilot1 and pilot2, respectively.

Remove_final_characters.txt : code used in PowerShell window (in Windows) to remove en masse final characters from the audio files of each participant; this is needed in order to be able to run the Praat scripts for audio annotation

REPSWITCH_Audio_files_organise.rmd : code to better organise the information in the Praat output files and to merge them with the respective participant's PsychoPy output file

REPSWITCH1_Clean.rmd : a working file I was using for different purposes

Results_Pilots1&2.pptx : presentation of the results from pilots 1 and 2 (figures)
